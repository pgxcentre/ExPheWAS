This data release was presented at the International Genetic Epidemiology Society and the American Society for Human Genetics 2021 meetings.

Association testing was done using UKBPheWAS v2.0

Changes from the previous data release (v0)

  • Sex-stratified analyses are provided
  • Interactive cis-MR analyses
  • We now include lincRNA genes and protein coding genes (Ensembl 87)
  • Automatic enrichment analysis using g:Profiler (outcome page)
  • We used asymetric padding of 1.5kb on the 5' end and 1.0kb on the 3' end
  • Phecodes were used instead of the raw ICD10 codes
  • Genes on chromosome X
As part of our commitment to make ExPheWAS results stable, the data release v0 is still available.
